## Password

Email-based accounts can change the password from **Profile → Security**. Enter the current password and a new password that passes the strength checks. Password reset is also available from the sign-in page by email link.

Repeated failed sign-ins can temporarily lock the account. Contact support if you cannot recover access.

## Google connection

Use **Profile → Connections** to link or unlink Google. Linking gives the same account an additional sign-in method; it does not create a second member.

## Two-factor authentication

TOTP-based two-factor authentication is currently available to Admins and Managers.

<Steps>
  <Step title="Enroll">
    Open **Profile → Security** and click **Enable 2FA**. Scan the QR code with a TOTP authenticator.
  </Step>
  <Step title="Verify">
    Enter the current six-digit code to complete enrollment.
  </Step>
  <Step title="Disable when required">
    Enter your account password to confirm disabling 2FA.
  </Step>
</Steps>

## New-device verification

Protected sign-ins can require an email verification code for an unfamiliar device. Enter the code sent to your account email. If **Remember this device** is selected, Flotick stores a trusted-device token in that browser to reduce later challenges.

Signing out or clearing browser storage can remove local trust. Flotick does not currently expose a client-facing list of trusted devices.

## Change a password

<Steps>
  <Step title="Open Security">
    Go to **Profile → Security** and choose the password action.
  </Step>
  <Step title="Confirm your current password">
    Enter the current password, then a new password that meets the strength requirements.
  </Step>
  <Step title="Save">
    Flotick verifies the current password and updates the account. Sign in again where prompted after the change.
  </Step>
</Steps>

Use the **Forgot password** link on the sign-in screen when you cannot provide the current password. The reset email is time-limited. If it does not arrive, check spam and confirm that you are using the email associated with your Flotick account.

## Set up two-factor authentication

Install a standard TOTP authenticator first. During enrollment, scan the QR code, then enter the current code generated by the app. Keep the authenticator available before you sign out. Disabling two-factor authentication requires your current account password, so an unauthorized person cannot remove the second factor without it.

## Keep sign-in safe

Never share a password, reset link, verification code, or TOTP code. If you suspect compromise, change the password, review Google connection status, and contact [support](/resources/support). When using a shared device, do not select **Remember this device** and sign out when finished.
