Tune Flotick's notifications to match how you want to be reached. You'll find these controls in **Profile → Notification preferences**.


## The matrix

For each notification type, choose which channels deliver it:

- **In-app** — bell, Inbox, and live toast
- **Email** — sent to your account email
- **Telegram** — only if you've connected Telegram (see [Telegram](/features/telegram))

## Common types to consider

| Type | Sensible default |
| :--- | :--- |
| Task assigned to me | All channels |
| Mention in a comment | All channels |
| Status changed on my task | In-app + email |
| Sprint started or completed | In-app |
| Leave approved or rejected | In-app + email |
| Payroll run finalized | In-app + email |
| AI insight (high priority) | In-app + email + Telegram |
| Files shared with me | In-app |

Defaults are reasonable; tune as you go.

## Mute conversations

Where supported, a comment thread shows a **Mute** action — useful for high-traffic conversations you don't need to follow.

## Email volume

Email is reserved for **High** and **Urgent** priorities by default. If your inbox is too quiet (or too loud), adjust by type rather than turning email off entirely.

## Tips

- Check the AI Insights toggle. Many users don't realize Insights can land in their email.
- Telegram is great for urgent items only. Resist the urge to mirror everything — it leads to muting the bot.
- If you take time off, briefly disable email for low-priority types so you can focus on the in-app catch-up later.
