## Read the matrix correctly

The matrix describes product entitlement, not a promise that every member can use every listed feature. A member must also have an active organization account, the required role or override, and access to the relevant Project or data.

For example, Pro enables Env Storage for the organization, but a Member still needs its Env Storage permission and access to the selected Project. Max enables scheduled reports, but only Admins and Managers can create and manage them.

## Temporarily unavailable modules

Flotick can hide fully built modules during pre-launch. The matrix documents the client-facing modules currently enabled in the application. If a feature is not visible in your sidebar or settings, do not assume it is available solely because a plan supports related capability.
