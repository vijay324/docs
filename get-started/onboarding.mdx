Every new account walks through a short onboarding wizard. The wizard saves progress as you go, so you can leave and resume.

[IMAGE: Onboarding wizard step indicator with Profile, Preferences, and Done steps]

## What you'll set up

<Tabs>
  <Tab title="If you created a new organization">
    1. **Profile** — first name, last name, job title, mobile, profile picture
    2. **Organization details** — work mode (Office, Remote, Hybrid), standard work hours, optional logo
    3. **Invite teammates** (optional) — add emails to send invitations now
    4. **Confirm** — finish onboarding and land on the dashboard
  </Tab>

  <Tab title="If you accepted an invitation">
    1. **Profile** — first name, last name, job title, mobile, profile picture
    2. **Preferences** — light or dark theme, notification opt-ins
    3. **Introduction** (optional) — short bio shown to teammates
    4. **Confirm** — finish onboarding and land on the dashboard
  </Tab>
</Tabs>

## Skipping steps

Most steps are optional. Skipping them still marks onboarding complete. You can update any of these fields later from your [profile](/account/profile) or, if you're an Admin, from [organization settings](/admin/organization-settings).

## Re-running onboarding

If you'd like to walk through the introduction tour again, open your profile and click **Tour again**.
