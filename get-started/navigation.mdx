The sidebar groups personal work, delivery, team operations, files, insights, and role-specific administration. Available destinations adapt to role, plan, and individual permission.

## Search and commands

Press <kbd>Ctrl</kbd> + <kbd>K</kbd> on Windows or <kbd>⌘</kbd> + <kbd>K</kbd> on macOS. The command palette finds destinations, accessible work, and available creation actions.

| Shortcut | Action |
| --- | --- |
| `Ctrl/⌘ K` | Open command palette |
| `Ctrl/⌘ N` | Create a Task |
| `?` | Open shortcut help |
| `G` then `H` | Go to Inbox |
| `G` then `T` | Go to Tasks |
| `G` then `S` | Go to Sprints |
| `G` then `P` | Go to Projects |
| `G` then `B` | Go to My Board |
| `G` then `M` | Go to Team Members |
| `G` then `A` | Go to Analytics |

Navigation shortcuts do not run while you are typing in a field.

## Use the Project selector

Choose **All Projects** for cross-Project triage or select one Project before using a Project-specific board, backlog, Analytics view, Env Storage workspace, or recurring schedule. Pages that require one Project prompt you to choose it.

## Other navigation

The resource area links to documentation, Help, feature requests, and the changelog. Your profile menu contains profile, appearance, Trash, and sign-out actions. Use the sidebar footer to choose **System**, **Light**, or **Dark** appearance.

If the connection drops, Flotick shows its offline state and reconnects when the network returns.
