description: "Move through Flotick with the sidebar, command palette, and keyboard shortcuts."
---

The sidebar groups work into **Inbox**, **Tasks**, **Sprints**, **Projects**, tracking, storage, and role-specific administration. Use the Project selector to scope supported pages to one Project or all Projects.

## Search and commands

Press <kbd>Ctrl</kbd> + <kbd>K</kbd> on Windows or <kbd>⌘</kbd> + <kbd>K</kbd> on macOS. The command palette finds pages and available actions, including creating Tasks, Sprints, and Projects.

## Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl/⌘ K` | Open command palette |
| `Ctrl/⌘ N` | Create a Task |
| `?` | Open shortcut help |
| `G` then `H` | Go to Inbox |
| `G` then `T` | Go to Tasks |
| `G` then `S` | Go to Sprints |
| `G` then `P` | Go to Projects |
| `G` then `B` | Go to My Board |
| `G` then `M` | Go to Team Members |
| `G` then `A` | Go to Analytics |

Shortcuts do not run while you are typing in a field, except the command palette shortcut.

## Display and connectivity

Use the sidebar footer to select **System**, **Light**, or **Dark** appearance. If your connection drops, Flotick displays its offline state and reconnects automatically when the network returns.

## Sidebar areas

The main navigation gives you direct access to Inbox, Tasks, Sprints, Projects, Attendance, Leave, Analytics, All Files, and Env Storage. Admins and Managers also see team, holiday, and shift areas when their permissions allow them. Some pages are unavailable on plans that do not include the module.

The resource area links to changelog, documentation, Help, and feedback. Your profile menu contains account controls, appearance settings, sign out, and the feedback action.

## Use the Project selector

Several work-management pages use the Project selector in the app shell. Choose **All Projects** for cross-Project triage, or choose one Project before opening a Project-specific board, backlog, analytics, environment, or recurring-task view. When a page needs one Project, Flotick asks you to select one instead of guessing.

## Command palette behavior

The command palette only presents actions you are currently allowed to use. For example, an unavailable Project-creation action is not a sign-in problem; it may be restricted by role, permission, plan, or organization state.
