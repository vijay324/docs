Workflow customization is available on Pro and Max. Admins manage it from **Settings → Workflow configuration**.

## What you can configure

| Area | Configuration |
| --- | --- |
| Task statuses | Color for each fixed workflow status |
| Priorities | Color for `Critical`, `High`, `Medium`, and `Low` |
| Story points | Fibonacci, linear, or T-shirt estimation scale |

The status pipeline, priority keys, labels, and order remain fixed so boards, permissions, and Analytics stay consistent. Individual Tasks keep their separate three-stage workflow.

Use the live preview to review Task cards and board columns. Click **Save Changes** to publish the new appearance, or **Reset** to restore the default configuration.

Departments are managed from the separate **Departments** tab. See [Departments](/admin/departments).
