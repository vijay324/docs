## Core limits

| Limit | Free | Pro | Max |
| --- | ---: | ---: | ---: |
| Members | 5 | 25 | Unlimited |
| Active Projects | 3 | 15 | Unlimited |
| Storage | 500 MB | 5 GB | 100 GB |
| Maximum file size | 25 MB | 100 MB | 500 MB |
| General AI generations / month | 20 | 500 | Unlimited |
| Orbit Agent requests / month | 50 | 200 | Unlimited |
| Analytics ranges | 7 and 30 days | Up to 1 year | Up to 1 year |

Pro adds team analytics, exports, workflow customization, Task file uploads, Env Storage, team attendance, advanced leave management, model selection, and per-member permission overrides. Max removes core scale caps and adds scheduled reports.

Limits are organization-wide unless a page states otherwise. The app shows current usage in settings and blocks a new action when it would exceed the active entitlement.

<Note>
Flotick may adjust plan packaging before general availability. The pricing page is authoritative for what can be purchased.
</Note>

## How limits are applied

Flotick checks plan entitlement when you take an action, not only when you open a page. For example, it checks seats when an invitation is accepted, active-Project capacity when a Project is created or restored, file size before upload, and AI allowance before a generation begins.

An unlimited value removes the configured product cap; it does not bypass normal role, organization, Project, or security permissions.

## Compare before you choose

Choose **Free** for a small team using core delivery work, personal operations, and limited AI. Choose **Pro** when the organization needs team-level visibility, workflow control, richer people operations, Task attachments, environment configuration, or overrides. Choose **Max** when scale caps and scheduled reports are important.
