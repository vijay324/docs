---
title: "Leave policy"
description: "Configure leave types, allocation, accrual, and request rules."
---

Admins manage leave policy from **Settings → Attendance and leave policies**. Policy customization is available on Pro and Max.

Full-time employees and interns have independent policies. Contract employees are excluded from leave policy.

## Available controls

- Enable or disable each leave type
- Set its default annual allocation
- Enable accrual and choose monthly, quarterly, half-yearly, or annual cadence
- Set days added per accrual period
- Allow or block past-date requests
- Exclude or include weekends in the charged day count
- Set minimum advance notice
- Set maximum consecutive days

Save changes to apply them to new validation and future balance calculations. **Reset** restores Flotick's defaults.

<Warning>
Review active requests and member balances before changing enabled types, allocation, or accrual. Existing approved requests remain recorded.
</Warning>

Administrative leave tools can run accrual and carry-forward and record adjustments, comp-off decisions, and encashment transactions on eligible plans. Use balance history to verify the resulting ledger.
