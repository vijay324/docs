Open **Settings → Integrations** to manage external connections.


## Telegram

Org-level Telegram setup is a member-by-member action — each member connects their own account from their profile or from this page. See [Telegram](/features/telegram) for the connect flow.

The org-level configuration controls whether Telegram-based features (e.g., daily stand-up prompts via the bot) are enabled at all. Members still control which notification types they receive personally.

## Billing portal

If your org has an active subscription, the **Billing portal** link opens your hosted billing page where you can:

- Update the payment method on file
- Download invoices
- See plan history

For plan changes (upgrade, downgrade, cancel), the in-app actions in **Admin → Billing** are typically faster. See [Upgrade or cancel](/admin/billing/upgrade-or-cancel).
