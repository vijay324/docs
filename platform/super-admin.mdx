---
title: "Super Admin"
description: "Global operations console for platform owners running multiple tenant organizations."
---

The super-admin surface is separate from tenant administration and is intended for trusted platform operators.

## Routes

- `/super-admin/login` and verification flow
- `/super-admin/dashboard`
- `/super-admin/organizations`
- `/super-admin/users`
- `/super-admin/subscriptions`
- `/super-admin/analytics`
- `/super-admin/infrastructure`
- `/super-admin/audit`

## API surface

All backend operations are under:

```txt
/api/v1/platform-admin/*
```

## Security model

- Dedicated auth middleware and permission checks.
- IP allowlist enforcement on protected operations.
- 2FA verification flow for privileged sessions.

## Best practices

- Grant least-privilege permissions.
- Monitor audit logs continuously.
- Keep platform-admin credentials and policies isolated from tenant auth processes.
