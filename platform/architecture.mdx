## API surfaces

- `/api/v1/*` — versioned REST (the primary API).
- `/api/v1/o/:orgSlug/*` — org-scoped aliases that resolve organization membership before handlers run.
- `/api/auth/*` — authentication endpoints (JWT-based).
- `/api/v1/graphql` — an optional GraphQL endpoint for selected aggregated reads.

## Authentication

- **Members** use JWT access tokens (Bearer) plus an HttpOnly refresh-token cookie with family rotation.
- Refresh tokens are rotated; reuse of a rotated token revokes the whole token family.

## Realtime & background work

- Socket.io delivers live updates, scaled across instances via a Redis adapter.
- Server-Sent Events stream long-running flows such as Orbit Agent execution.
- Background jobs (queues + cron) handle analytics, reminders, trash cleanup, and plan enforcement.

## Security & observability

- Field-level **AES-256-GCM** encryption at rest for sensitive content (e.g. task descriptions).
- TLS in transit, Helmet headers, CORS, and layered rate limiting.
- Health checks (`/health`, `/api/v1/health`), Prometheus metrics, and correlation IDs for tracing.
