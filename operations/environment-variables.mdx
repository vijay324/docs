---
title: "Environment Variables"
description: "Production-ready environment configuration for frontend and backend runtimes."
---

## Required backend variables

At minimum:
- `MONGODB_URI`
- `JWT_SECRET`
- `BETTER_AUTH_SECRET`
- `FRONTEND_URL`
- `BACKEND_URL`
- `CORS_ORIGIN`

Additional production-critical values:
- `REDIS_URL`
- `SECURITY_FAIL_CLOSED=true`
- `OPS_API_TOKEN` (for protected ops endpoints)

## Frontend runtime variables

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_BACKEND_URL`
- `NEXT_PUBLIC_APP_URL`

## Configuration rules

- Keep secrets out of `NEXT_PUBLIC_*`.
- Enforce HTTPS URLs in production.
- Validate variable completeness before deployment rollout.

<Warning>
  Backend startup may fail intentionally when required values are missing in production mode.
</Warning>
