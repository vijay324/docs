Analytics adapts to role and plan. Members receive personal delivery analytics. Admins and Managers can access organization and Project views when the active plan enables them.

## Sections

| Section | Examples |
| --- | --- |
| Overview | Completed work, completion rate, cycle time, activity, anomalies, and executive brief |
| Projects | Active Projects, status, blocked and overdue work |
| Team | Completed work, review and testing load, member drill-down |
| Sprints | Velocity, predictability, cumulative flow, and cycle trends |
| Tasks | Status, flow, quality, age, and movement |
| Attendance | Attendance rate, hours, late arrivals, and work location |
| Leave | Leave volume and type |
| Deadlines | On-time delivery, overdue work, slippage, and risk |
| Workload | Open work by member and workload heatmap |

Use the Project selector and time-range control to change scope. Supported ranges depend on plan: Free offers 7 and 30 days; paid plans also offer 90 days and one year.

CSV export is available when the plan enables data export. Refresh clears the current analytics cache and requests updated data.

<Note>
Some cards require enough history to calculate a meaningful result. Flotick shows an empty or unavailable state instead of estimating missing data.
</Note>

## Navigate Analytics

Use the primary tabs to move between Overview, Projects, Team, Sprints, Tasks, Attendance, Leave, Deadlines, and Workload. The available tabs adapt to role: personal views focus on your delivery; leader views add team, workforce, and Project drill-down. A Project selector scopes supported analytics to a specific Project.

## Read the metrics

Analytics includes activity counts, completed work, completion rate, active members, status breakdowns, cycle-time percentiles, work-in-progress age, blocked age, velocity, predictability, workload distribution, deadline slippage, and due-date risk. Metric cards link to richer charts or drilldowns where available.

Use time ranges consistently when comparing periods. A short range is useful for daily execution; 90 days or one year is better for recurring delivery patterns when your plan includes those ranges.

## Export and refresh

Eligible paid plans show **Export CSV** for supported organization data. Exports reflect the current selected scope and filters. Use **Refresh** when you need a new server snapshot rather than the current cached view.

## Interpret responsibly

Metrics show work recorded in Flotick; they do not by themselves measure effort, performance, or quality. Review workload, status, time, leave, and project context together before making people or staffing decisions.
