---
title: "Work Management"
description: "Projects, tasks, sprints, board workflows, and analytics for execution teams."
---

## Projects

- Create and configure projects from `/{orgSlug}/projects`.
- Control visibility and membership.
- Track project-level delivery metrics and settings.

## Tasks

- Full task lifecycle in `/{orgSlug}/tasks`.
- Dedicated create/edit/detail views.
- Rich assignment, priority, due-date, comments, and worklog flows.
- File attachments through `/api/v1/files/*`.

## Board

- Kanban workflow in `/{orgSlug}/board`.
- Project filtering and personal-board behavior.
- Drag-and-drop updates with realtime sync.

## Sprints

- Sprint creation, status, and details in `/{orgSlug}/sprints`.
- Burndown, velocity, and sprint member operations.
- Sprint settings route for lifecycle control.

## Workflow customization

- Organization-level workflow configuration in `/{orgSlug}/settings/workflow`.
- Custom statuses, transitions, priorities, and related task metadata.

## Analytics

- Delivery analytics in `/{orgSlug}/analytics`.
- Dashboard and project analytics endpoints for executive reporting.
