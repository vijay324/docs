## What it does

Dashboard gives each user a live snapshot of assigned work, activity, and team momentum.

## Why it matters

- Helps users start the day with the right priorities.
- Reduces status meetings by exposing progress in one place.
- Surfaces blockers and overdue work early.

## How to use it effectively

1. Open dashboard first at the start of every workday.
2. Review "My Tasks" and due dates before accepting new work.
3. Use quick actions to create tasks, sprints, or projects when needed.
4. Check attendance panel to maintain complete daily records.

## Productivity tips

- Keep task priorities current to improve dashboard accuracy.
- Use dashboard as the source of truth during standups.
