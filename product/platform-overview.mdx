## Key capabilities

- Multi-tenant route model using `/{orgSlug}/...` to keep user context and data boundaries explicit.
- Real-time collaboration powered by Socket.IO and canonical `realtime:event` envelopes.
- Dual admin surface: tenant admins/managers and platform super-admin operators.
- Unified API architecture with REST + GraphQL, plus org-scoped endpoint aliases.
- Production controls for rate limits, secure headers, metrics, health checks, and webhook verification.

## Navigation map

- `/{orgSlug}/dashboard`: executive and personal command center.
- `/{orgSlug}/tasks`, `/{orgSlug}/board`, `/{orgSlug}/sprints`, `/{orgSlug}/projects`: delivery workflows.
- `/{orgSlug}/attendance`, `/{orgSlug}/leave`, `/{orgSlug}/payroll`, `/{orgSlug}/employees`: workforce operations.
- `/{orgSlug}/settings/*`: policy, billing, and workflow customization.
- `/super-admin/*`: global operations and compliance.
