## How should we start using Flotick as a new team?

Start with one pilot team, define project and task workflows, then roll out to additional teams after one sprint.

## Who should own workspace setup?

An admin should own setup, with managers responsible for day-to-day workflow consistency.

## How often should users update tasks?

At minimum once per day. Real-time updates are best for team visibility.

## Should we use board or list view?

Use board view for flow and bottlenecks, and list view for bulk updates and detail-heavy work.

## How do we improve adoption?

Make Flotick the default for standups, planning, and status reviews. Leadership habits drive team behavior.

## How do we keep payroll runs accurate?

Ensure attendance and leave records are complete before payroll execution and review anomalies before finalization.
