description: "At-a-glance view of core features, primary users, and outcomes."

| Feature | Best for | Primary value |
| --- | --- | --- |
| Dashboard | Everyone | Daily focus and shared visibility |
| Projects | Admin, Manager | Clear ownership and initiative structure |
| Tasks | Everyone | Action-level accountability |
| Board | Everyone | Fast progress tracking and blocker detection |
| Sprints | Engineering teams | Better planning and predictable delivery |
| Analytics | Admin, Manager | Data-driven planning and improvement |
| Attendance | Everyone | Reliable daily work records |
| Leave | Everyone, Manager | Transparent planning and approvals |
| Payroll | Admin, Manager | Structured compensation operations |
| Employees | Admin, Manager | Role clarity and team management |
| AI Insights | Admin, Manager | Faster summaries and planning support |

## Role guidance

- **Admin**: full organization control.
- **Manager**: team operations and delivery oversight.
- **Employee**: daily execution and self-management workflows.
